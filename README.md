# Teste de vaga fullstack *KRONOOS*

## Descrição

Conversão de um arquivo .csv para array de objetos.

## Tech Stack

*  [Node.js](https://nodejs.org/en/)
*  [Jest](https://jestjs.io/)
*  [csv-parser](https://www.npmjs.com/package/csv-parser)

## Rodar Aplicação
```
npm ci
npm run dev
```

## Rodar Testes
```
npm run test
```
